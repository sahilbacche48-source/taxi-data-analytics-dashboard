# 🚕 Taxi Data Analytics & ML Dashboard

## 📌 Project Overview
This project analyzes NYC taxi trip data using PySpark and builds a machine learning model to predict fare amount. A Power BI dashboard is created for visualization.

## 🔧 Tools Used
- PySpark
- Python
- Machine Learning (Linear Regression)
- Power BI

## 📊 Features
- Data cleaning and preprocessing
- Predictive analysis (fare prediction)
- Interactive dashboard with insights

## 📈 Dashboard Insights
- Trip patterns
- Passenger trends
- Revenue distribution
